# 🚀 Airdrop Payout Dashboard

A modern crypto reward dashboard for managing and tracking airdrop payouts.

- Wallet connection interface
- Airdrop task system
- Referral tracking
- Clean responsive UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 📦 Deployment
Hosted on Netlify via GitHub.
